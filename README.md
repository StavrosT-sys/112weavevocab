# VocabWeave

Enhanced English vocabulary learning with adaptive spaced repetition - 100% Oxford 3000. Client-side only, mobile-first for Brazilian Portuguese speakers.

## Setup
1. Clone: `git clone https://github.com/StavrosT-sys/VocabWeave.git`
2. Install: `npm install`
3. Dev: `npm run dev` (localhost:5173)
4. Build: `npm run build` (dist/ folder for deploy)

## Features
- Semantic graph: Tap nodes to weave word paths (Framer Motion).
- Quests: Unlock badges with progress (LocalStorage).
- Duels: Share canvas challenges via URL.
- FSRS: Stability viz (25% better than SM-2).
